# Entrega Final – Proyecto React (TT 2025)

Descripción
Aplicación desarrollada en React que simula un e-commerce de productos deportivos.
Incluye carrito de compras, autenticación de usuarios, CRUD de productos con MockAPI,
búsqueda, paginación y despliegue en GitHub Pages.

1- Funcionalidades principales
    - Carrito de compras
    - Gestión global con Context API
    - Agregar, sumar, restar, eliminar y vaciar productos
    - Diferentes fuentes de productos (MockAPI y FakeStore)
    - Checkout protegido por autenticación
    - Notificaciones visuales con React Toastify

2- Autenticación
    - Login simulado con localStorage
    - Rutas protegidas (Checkout y Administración)
    - Redirección automática luego del login

3- CRUD de Productos (MockAPI)
    - Crear, leer, editar y eliminar productos
    - Formulario controlado con validaciones
    - Modal de confirmación al eliminar
    - Manejo de estados de carga y error
    - Feedback visual con Toastify

4- Búsqueda y Paginación
    - Búsqueda en tiempo real por nombre/categoría
    - Paginación de productos
    - UX optimizada (reinicio de página al buscar)

-Tecnologías utilizadas:
    - React
    - React Router
    - Context API
    - Bootstrap
    - React Toastify
    - MockAPI
    - GitHub Pages
